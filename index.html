Process

How Automated
Permit Processing Works

Standard mechanical work — changeouts, replacements, like-for-like installs — follows a predictable pattern. HVACapp turns that pattern into a standardized application that can be checked and processed without weeks of back-and-forth.

STEP 1
Enter system specs

Submit equipment specifications, location, and project details through a standardized application — the same fields every time, no jurisdiction guesswork.

STEP 2
Automated compliance checks

Each data point is checked against mechanical code and Title 24 requirements, so the application that reaches the jurisdiction is complete and compliant.

STEP 3
Permit processed and tracked

Compliant applications move straight to filing — a licensed permit runner shepherds it through the jurisdiction and you track status until it's approved.

Example — License Lookup
Contractor enters license number t + 0s
System checks cache, falls back to a live Livermore lookup if needed t + 2s

Compliance Is at the Core

Every application is validated against current mechanical code and Title 24 energy requirements before it reaches the jurisdiction — so what gets filed is what gets approved.

Both Sides Benefit

Contractors get permits moving in hours instead of weeks. Jurisdictions get complete, standardized applications instead of incomplete paperwork that eats staff time.

Works How You Work

Submit from the job site or the office. No new software for the jurisdiction to adopt — applications arrive through the permitting systems cities already use.

The Unpermitted-Work Gap

New installs almost always get permitted; changeouts are where paperwork slips. HVACapp makes the permit as fast as the job, so compliance stops being the bottleneck.

Contractors — Reviewed Faster, Installed Faster

Available to CSLB-licensed C-20 contractors. Standardized applications and automated compliance checks mean faster time from sale to install — and a permit history lookup for any license, any time. Grow the business instead of chasing paperwork.
